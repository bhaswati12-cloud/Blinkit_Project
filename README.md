 📊 Blinkit Power BI Dashboard Project

Welcome to the **Blinkit Power BI Dashboard** project — a comprehensive and interactive business intelligence dashboard designed to uncover insights from Blinkit's retail data.

🔍 Project Overview

Blinkit, India’s last-minute delivery app, operates in a fast-paced, hyperlocal retail environment. This dashboard was developed to analyze key operational and sales metrics to help stakeholders make informed decisions.

 🎯 Objectives

- Track **Total Sales**, **Average Sales**, **Item Visibility**, and **Customer Ratings**
- Analyze performance across **outlet sizes**, **location tiers**, and **item types**
- Enable users to explore data using interactive filters and visuals
- Identify trends and patterns to support business growth and efficiency

 Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (ETL)
- Custom Visuals & Filters

 📈 Key KPIs

- 💰 Total Sales: $1.20M  
- 🛒 Average Sales per Item: $141  
- 🌟 Average Ratings: 3.9  
- 📦 Total Number of Items Sold: 8,523  

 📊 Dashboard Features

- Line Chart: Sales trend over years to show outlet growth and seasonality  
- Donut Charts: Breakdown of sales by outlet size and fat content  
- Bar Charts: Sales by item category and location tier  
- Matrix Table: Summary by outlet type with key metrics like sales, visibility, ratings  
- Interactive Filters: Slice data by city tier, item type, and outlet size  

📌 Insights

- Tier 3 outlets generate the highest sales, despite fewer outlets  
- Low Fat items contribute significantly in Tier 2 regions  
- Supermarkets outperform other types in both visibility and product diversity
